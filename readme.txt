== Description ==

This a developer friendly plugin but can also be used via shortcodes <strong>[smt_facebook]</strong> <strong>[smt_twitter] </strong> <strong>[smt_instagram]</strong>
You the option for two templates:

<ol>
<li>Default</li>
<li>Developer</li>
</ol>

 You can change the templates by setting the attribute template ='developer'
 For instance if you want to change the template of facebook to developer all you would have to do is [smt_facebook template="developer"]

 You can also customize the template as well. Just drag the "Template" folder from the plugin into your theme and rename it to social-meida-trifecta.
 That should be about it
